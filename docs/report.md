## Вид заметки

Для заметки сохраненной в `.md`

```markdown
# Что такое анализ ? 14/5/21 @ 14:45:47
> Tags: Research

Анализ - понижение уровня абстракции
```

Для заметки сохраненной в `.txt`

```
Title: Что такое анализ ?
Date: 14/5/21 @ 14:49:46
Tags: Research
Анализ - понижение уровня абстракции
```
